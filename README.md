# SimpleAnalysis

Проект по ознакомительной практике: анализ набора случайных числовых данных средствами Python.

Выполняемые операции:

- генерация 1000 целых чисел в диапазоне от -10000 до 10000;
- создание объекта `pandas.Series`;
- очистка данных от пропусков, нечисловых значений и значений вне допустимого диапазона;
- расчет минимума, максимума, количества повторяющихся значений, суммы и среднеквадратического отклонения;
- сохранение исходного набора в CSV и TXT;
- построение линейного графика, гистограммы с математическим округлением до сотен и совместного графика отсортированных рядов;
- автоматическое модульное тестирование.

## Структура

- `simple_analysis.py` - исполняемый модуль;
- `tests/test_simple_analysis.py` - модульные тесты;
- `requirements.txt` - список зависимостей для установки через `pip`;
- `docs/cleaning_algorithm.drawio` - блок-схема очистки данных в формате draw.io;
- `docs/processing_algorithm.drawio` - блок-схема обработки данных в формате draw.io;
- `docs/cleaning_algorithm.dot` - дополнительный исходный файл блок-схемы очистки данных.

## Подготовка среды

1. Установить Python 3.11 или новее.
2. В PyCharm создать проект с виртуальным окружением.
3. Установить библиотеки NumPy, pandas и Matplotlib:

```text
python -m pip install -r requirements.txt
```

## Запуск

```text
python simple_analysis.py
```

После запуска программа создаст `dataset.csv`, `dataset.txt`, `results.txt` и каталог `figures` с тремя графиками.

## Проверка

```text
python -m unittest discover -s tests -v
```

Тесты проверяют генерацию данных, очистку, вычисления, правило округления, сортировку и формирование файлов результата.
